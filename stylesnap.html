stylesnap

An AI-powered product content tool that automates photo editing for retail sellers.

Overview

StyleSnap is a solo project that streamlines the product photo workflow for online sellers. By automating background removal, lighting correction, and image optimization, it reduces the need for manual editing - saving time and improving consistency across platforms like Amazon, Etsy, and Shopify.

My Role

User Research / Persona Creation / Visual Design / Prototyping

  • Conducted surveys and in-depth user interviews
  • Created user personas and journey maps
  • Designed low to high-fidelity prototypes
  • Iterated based on usability testing
  • Developed key visuals for product identity

Key Features

  • Automated Editing - AI removes backgrounds, adjusts lighting, and formats for upload
  • Time-Saving Workflow - Cuts repetitive editing for multi-platform sellers
  • Scalable Output - Designed for bulk image generation with uniform styling
  • Platform Optimization - Auto-formats images for Amazon, Etsy, Shopify, etc.
  • Consistent Branding - Delivers polished, professional visuals for any shop size
